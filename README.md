# 在线证书验真系统

这是一个用于验证计量证书真实性的在线系统，支持中英文切换。

## 功能特点

- 证书信息展示
- 中英文双语切换
- 二维码显示
- 徽章展示

## 部署方式

1. 在GitHub上创建一个新仓库
2. 将所有文件上传到仓库的 `main` 分支或 `gh-pages` 分支
3. 在仓库设置中开启GitHub Pages，选择相应分支

## 文件结构

```
.
├── index.html          # 主页面
├── images/             # 图片资源
│   ├── gzjl.png        # 二维码图片
│   └── tubiao.png      # 徽章图片
├── .gitignore          # Git忽略文件
└── README.md           # 项目说明
```
